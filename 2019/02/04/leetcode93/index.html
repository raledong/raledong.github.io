leetcode93.Restore IP Addresses

发表于 2019-02-04 更新于 2024-08-18 分类于 面试 ， leetcode

题目要求

返回字符串能够组成的所有IP地址。

思路与代码

IP地址由0/1位二进制数字构成，一共分为4个区间，每个区间8位。因此每个区间的值转化到十进制为0~255。那么我们只要划分出这四个区间，然后判断这四个区间的值是否符合标准即可。
