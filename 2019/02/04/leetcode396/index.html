题目要求

Bk代表对数组A在位置k上进行顺时针的旋转后生成的数组。F(k) = 0 * Bk[0] + 1 * Bk[1] + ... + (n-1) * Bk[n-1]，要求返回获得的最大的F(k)的值。

暴力循环

按照题目的要求，执行两次循环即可以获得F(k)的所有值，只需要从中比较最大值即可。
